### Create a new demo

To create a brand new demo, you can create a new net demo using Lancey’s chrome extension or you can choose to clone a demo based on what one of your team members have created.

For the purpose of this guide, we’ll focus on cloning existing demos as we can speed up the process.

Within the Lancey dashboard, you’ll notice there are demos that are already created by your team. Let’s learn to clone one.

1. Find a demo you want to clone    
2. Click on the demo name head to the demo editor
3. Once you’ve opened up a demo, click on **Copy demo** button to create a copy for your own use case    
4. Once your demo is cloned, make sure to navigate to the new demo so you don’t modify someone else’s by mistake!
