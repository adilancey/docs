# **How to install the Chrome Extension**

This guide covers how to install the Lancey Chrome extension in just a few clicks so you can start creating interactive demos!

Before you begin, make sure you've signed up for an account. If you haven't, head over to https://lancey.io/pricing and get started for free!

# **Step 1: Head to the [Chrome web store](https://chrome.google.com/webstore/detail/lancey/alplbnljhpemnggmbiknaacdjfdookdj)**

Once you're on the page, click on the **Add to Chrome**

!https://d33v4339jhl8k0.cloudfront.net/docs/assets/63a0bb4c1e9b0d75c72746e6/images/63b4606b0e47b51030546355/file-RERAtMsUpP.png

**⚠️ Please note:** Lancey's Chrome extension is **not compatible with other browsers** such as Firefox or Safari.

# **Step 2: Pin the extension**

It's possible it's hidden behind the puzzle icon 🧩. This is Chrome's way to let you choose which extension are visible. Here's how to check:

1. Click the **puzzle icon** to reveal all your Chrome extensions.
2. Find **Lancey** and press the **pin icon**. This will move your Lancey extension to your Chrome taskbar for easy future access.

!https://d33v4339jhl8k0.cloudfront.net/docs/assets/63a0bb4c1e9b0d75c72746e6/images/63b47665c21b4601058955c7/file-XTPazqgW92.png

# **Step 3: And that's it! 🎉 You're ready to create your first demo.**

# **Troubleshooting the extension:**

If you're facing issues with the Chrome extension, here are few steps to troubleshoot.

- Clear your cookies for the site [app.getlancey.com](http://app.getlancey.com/)

!https://d33v4339jhl8k0.cloudfront.net/docs/assets/63a0bb4c1e9b0d75c72746e6/images/63c980332b293e06999c4563/file-Fyaulhe5Za.png

- Uninstall the Lancey extension
- Reinstall the extension

If the issue persists, please contact Support.
