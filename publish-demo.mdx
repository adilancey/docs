## Publishing a demo

All demos created in Lancey get saved automatically but don't get published until you publish them. This is important because it allows you to continue making tweaks and updates without impacting what demo viewers see.

### Publishing options

When you have pending changes to publish, you'll need an alert in the top right bar next to the Preview button.

- You can open the **Publish** menu by clicking on the Publish button.

- You'll get to see when the demo was last published (or if you've never published it previously).
- Hit the Publish button to make the demo public. This means everyone with the link to this demo can view it.
