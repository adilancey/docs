---
title: "Github"
description: "How to set up the Github integration."
---

Lancey needs access to your repositories in order to work.

1. Head to [Integrations](https://platform.lancey.ai/settings/integrations/github.com) page
2. Click on**GitHub ** connector  
3. Complete the login flow.
4. Choose _all_ or specific repos that you want to connect to Lancey.
5. You will be redirected back to Lancey. If not, try refreshing the page.

Once connected, you’ll see an option to select a **repo ** where you want Lancey to work with, and a prompt input box.
