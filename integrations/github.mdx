---
title: "Github"
description: "How to set up the Github integration."
---

Lancey needs access to your repositories in order to work.

<Steps>
  <Step title="Integration settings">
    Head to [Integrations](https://platform.lancey.ai/settings/integrations) page
  </Step>
  <Step title="Connector">
    Click on **GitHub** connector 
  </Step>
  <Step title="Login flow">
    Complete the login flow.
  </Step>
  <Step title="Authorizing a repo">
    Choose all or specific repos that you want to connect to Lancey. You will be redirected back to Lancey. If not, try refreshing the page.
  </Step>
  <Step title="Selecting a repo" stepNumber={5}>
    Once connected, you’ll see an option to select a **repo**  where you want Lancey to work with.
  </Step>
</Steps>
